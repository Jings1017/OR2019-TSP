# OR2019-TSP

## Describe
Using Simulated Annealing algorithm to implement the Travel Salesman Problem</br>

## Command
$python3 tsp.py</br>

## Output 
total distance </br>
the optimal solution by SA</br>
total time</br>

## Example 
```
total distance = 2617 
7 -> 13 -> 11 -> 14 -> 2 -> 10 -> 4 -> 5 ->
8 -> 15 -> 12 -> 3 -> 1 -> 6 -> 0 -> 9 -> 7
 ---- running time : 10.77985572819414 seconds ----
 ```
