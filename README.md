# OR2019-TSP

## Describe
using Simulated Annealing algorithm to implement the Travel Salesman Problem

## Command
$python3 tsp.py

## Output 
total distance 
<the optimal solution by SA>
total time

## Example 
total distance = 2617

8 -> 15 -> 12 -> 3 -> 1 -> 6 -> 0 -> 9 -> 7
